About me

I am a Computational Chemistry student at the University of Oxford, graduating with a first-class honours degree in July 2014. I have three years experience in programming both for my Masters project and in my free time. My main expertise is in the Python programming language though I am also familiar with Java and web development using HTML, CSS, JavaScript and jQuery.

PyNonbonded

My most recent project was completed for my Masters thesis in Computational Chemistry. The project sought to describe forces between molecules using a forcefield-style set of empirical potentials. The source code is available from the GitHub repository and we also deployed a version to the cloud using Google App Engine which is hosted here.
